## 1. HTML 정의

(3)번인 Hyper Text Markup Language 이다.

## 2. HTML 개념

1) F : 웹 표준을 만드는곳은 WHATWG이다. Mozilla 재단이 속해있기는 하다.
2) T : table을 만들 때 행 의 개념이 \<th> 태그이기 때문에 반드시 존재해야 한다.
3) T : Heading 태그 자체에 어떠한 기능이 있는 건 아니지만 가독성을 위해 제목 이외에는 사용을 지양한다.
4) F : \<ol> 태그를 이용해서 리스트를 나열할 수도 있다.
5) F : 닫는 태그가 필요하지 않은 태그들도 존재한다. (br, img 등)
